REST API Documentation

Overview

This documentation provides information about the REST API endpoints, request methods, and response formats.

Endpoints

GET /api/users

Retrieve a list of all users.


    Request:
    GET /api/users

    Response:
    Status: 200 OK
    [
        {
            "id": 1,
            "name": "John Doe",
            "email": "john.doe@example.com"
        },
        {
            "id": 2,
            "name": "Jane Smith",
            "email": "jane.smith@example.com"
        }
    ]

POST /api/users

Create a new user.


    Request:
    POST /api/users
    Content-Type: application/json
    {
        "name": "New User",
        "email": "new.user@example.com"
    }

    Response:
    Status: 201 Created
    {
        "id": 3,
        "name": "New User",
        "email": "new.user@example.com"
    }

GET /api/users/{id}

Retrieve a user by ID.


    Request:
    GET /api/users/1

    Response:
    Status: 200 OK
    {
        "id": 1,
        "name": "John Doe",
        "email": "john.doe@example.com"
    }

PUT /api/users/{id}

Update a user by ID.


    Request:
    PUT /api/users/1
    Content-Type: application/json
    {
        "name": "Updated User",
        "email": "updated.user@example.com"
    }

    Response:
    Status: 200 OK
    {
        "id": 1,
        "name": "Updated User",
        "email": "updated.user@example.com"
    }

DELETE /api/users/{id}

Delete a user by ID.


    Request:
    DELETE /api/users/1

    Response:
    Status: 204 No Content
